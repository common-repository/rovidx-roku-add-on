== Description ==
Roku VOD Lite from **RoVidX** provides XML feeds compatible with the Roku VideoPlayer app example in the SDK.

Pro version includes Roku linking, extended category & leaf support and more.

[Get the Pro version here!](http://rovidx.com/downloads/roku-feeds-addon-pro/)

== Installation ==
1. Upload & Activate Plugin
2. Navigate to \"Roku Addon\" in the RoVidX administration menu
3. Follow the directions to setup your Roku app

== Frequently Asked Questions ==
= Will this work with HD? =

No, for HD you require a customized Roku application as well as the Pro edition of this plugin.   The Pro edition also includes a Roku App Template that utilizes RoVidX\'s full system.

= Will this work with a \"public\" channel or just \"private\"? =

This plugin works great with Public channels!  While your content will only be available in SD mode, the output from RoVidX meets the Roku SDK guidelines.

= Can I use my own custom app with Roku Addon? =

You sure can! But you will have to make some changes to the base code of your Roku app.

= What is the difference between the Roku Addon Lite & Roku Addon Pro versions? =

The free version only works with SD streams and has limited features compared to the Pro version.

Tech support is provided on a first-come first-serve basis for Free plugins.

Pro Editions include priority tech support as well.
